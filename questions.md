Write a program that determine the grade that someone
had using this criteria
* < 2.0 : Pass
* < 2.5 : Third
* < 3.0 : second class lower
* < 3.5 : Second class upper
* < 4.0 : First class

---
Operations for comparing:
* Equal: == 
* Less than: <
* Greater than: >
* Less than or equal to: <=
* Greater than or equal to: >=
* Not equal to: !=
